# Week 7 — Weather ETL Pipeline

A Python ETL pipeline that extracts real-time weather data from the 
OpenWeather API for 6 cities, transforms it into a structured format, 
and loads it into CSV and SQLite database.

## Cities Covered
Lagos, Abuja, London, New York, Tokyo, Paris

## Tools Used
Python, pandas, requests, sqlite3, matplotlib, seaborn

## Key Finding
Lagos was the hottest (24.4°C) and most humid (94%) city.
Tokyo was the only city experiencing Rain during data collection.
