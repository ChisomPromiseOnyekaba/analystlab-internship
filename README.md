# AnalystLab Africa Internship — Data Analytics Projects

## About
This repository contains all data analytics projects completed 
during my internship at AnalystLab Africa.

## Projects

### Week 1 — Online Retail Analysis (Python)
- Dataset: UCI Online Retail (541,909 transactions)
- Tools: Python, pandas, matplotlib, seaborn, VS Code
- Tasks: Data cleaning, EDA, visualisation, insights
- Key Finding: £8.9M revenue with 82% concentrated in the UK

### Week 2 — Netflix Titles Analysis (Python)
- Dataset: Netflix Titles (8,807 titles)
- Tools: Python, pandas, matplotlib, seaborn, VS Code
- Tasks: Data cleaning, EDA, visualisation, insights
- Key Finding: 61% of content targets adults; content grew 25x 2015–2019

### Week 3 — SQL Data Querying (PostgreSQL)
- Datasets: Chinook Music Store & Sales Data
- Tools: PostgreSQL, pgAdmin
- Tasks: Core SQL, JOINs, subqueries, window functions, business analysis
- Key Finding: Rock dominates Chinook revenue; Classic Cars lead sales

### Week 4 — COVID-19 Dashboard (Tableau)
- Dataset: Johns Hopkins University COVID-19 Time Series Data
- Tools: Python (pandas), Tableau Public
- Tasks: Data reshaping, interactive dashboard, trend analysis
- Key Finding: Omicron caused the largest single wave; USA led global cases

### Week 5 — Telecom Customer Churn Analysis (Power BI)
- Dataset: Telco Customer Churn (7,043 customers)
- Tools: Python (pandas, matplotlib, seaborn), Power BI
- Tasks: Business analytics case study, churn driver analysis, dashboard
- Key Finding: 26.5% churn rate; month-to-month contracts churn at 42.7%
